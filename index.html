    C

    C es un lenguaje de programación de propósito general originalmente desarrollado por Dennis Ritchie entre 1969 y 1972 en los Laboratorios Bell.Es un lenguaje imperativo de bajo nivel que proporciona acceso directo
    a la memoria y hardware del sistema

    Fuente:C(lenguaje de programación).

    Python

    Python es un lenguaje de programación interpretado, de alto nivel y de propósito general. Su filosofía de diseño enfatiza la legibilidad del código con el uso de sangría significativa. Sus constructores de lenguaje y su
    enfoque orientado a objetos tienen como objetivo ayudar a los programadores a escribir código claro y lógico

    Fuente:Python(lenguaje de programación)

    Javascript

    Javascript es un lenguaje de programación interpretado, de alto nivel, a menudo compilado just-in-time y multiparadigma. Tiene tipado dinámico, orientado a objetos basadas en prototipos y finciones de primera clase.
    Es uno de los tres lenguajes principales de la World Wide Web, junto a HTML y CSS

    Fuente:JavaScript
